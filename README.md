# License Serverless API (Vercel)

## Endpoint

GET /api/license?domain=example.com

## Alternatif (rewrite sayesinde)

GET /license?domain=example.com

## Deploy

1. Bu projeyi GitHub'a yükle
2. Vercel → New Project → Repo seç → Deploy
3. Test et:
   https://<project>.vercel.app/api/license?domain=example.com
